JDemetra+

JDemetra+ provides new Java implementations of TRAMO-SEATS and of X12-ARIMA.
Other time series problems, like modelling, benchmarking, temporal disaggregation are also considered.

JDemetra+, which is based on the NetBeans platform, is developed under the EUPL License.
